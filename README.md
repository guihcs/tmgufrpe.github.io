## Usage
To commit to this site you must be a member of the group. If you are not already included, please contact us.

### Run this repository
Clone the repository, and `cd` into it:
```sh
$ git clone https://github.com/ndrewtl/airspace-jekyll.git
$ cd airspace-jekyll
```

Install Dependencies:
```sh
$ bundle install
```

And run the server:
```sh
$ jekyll serve
```
